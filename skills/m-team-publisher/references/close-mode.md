# Close Mode

Use this file only when the current action is to close a task that has already been accepted as truly complete.

## Purpose

Close only when the task has reached its real terminal success state.

completed is not automatically closed.
Closure is the publisher's final quality decision.

## Close only when

- the goal is satisfied
- the evidence is sufficient
- no meaningful unresolved gap remains
- acceptance judgment is already clear

## Do not close when

- the latest baton merely sounds complete
- the files exist but do not prove the goal
- the chain still needs another baton
- you are only trying to tidy up the board

## Final reminder

Closing is a terminal action.
Do not use close as a substitute for reject.
If the task still needs work, reject it instead of closing it.
